מכוניות אוטונומיות וחשמליות

גבי מור

מכוניות אוטונומיות, משאיות שנוהגות את עצמן, עיר ללא רמזורים ותחבורה ציבורית ע״פ דרישה. מהפכת התחבורה החכמה כבר נמצאת הפתח ואיתה שלל הבטחות. בסקירה קצרה ועם קצת דמיון ננסה לצפות את השינויים ולנבא את אופן השפעתם על חיינו בשנים הקרובות.

גבי מור

ילד PC, החל את דרכו בעולם הפיתוח בגיל 14 עם שפות כמו פסקל ו C++. המשיך בשירות בממר"ם, ולאחר מכן במגוון סטארטאפים כמפתח fullstack, ראש צוות ויזם. כיום משלב בין אהבתו להוראה ולטכנולוגיה כמדריך בתחום ה front end, בקורסים למתחילים וכן בחברות מובילות (מייקרוסופט, נטאפ).

React Native - פיתוח Web פוגש את עולם הMobile

יריב גלעד

אחת הספריות הכי חמות בעולם הFED בשנתיים האחרונות היא React.js של פייסבוק. אחת החוזקות הבולטות של הספרייה היא העובדה שהיא מאפשרת למפתחים למנף את אותו ידע שרכשו בפיתוח Web בעולם הMobile. בהרצאה נבין למה הגישה החדשנית של React Native עשוייה להתגלות כ-Game Changer. בואו לגלות מדוע מותגים חזקים ומובילים כמו Instagram Airbnb ו Wix כבר משתמשים בReact Native באפליקציות הMobile שלהם.
